## 操作方法

- 上方向の移動: `W`もしくは`↑`(上矢印)
- 左方向の移動: `A`もしくは`←`(左矢印)
- 下方向の移動: `S`もしくは`↓`(下矢印)
- 右方向の移動: `D`もしくは`→`(右矢印)

- メーターの停止(ミニゲーム): `Space`

- アイテムの使用: `Q`, `E`, `R`(左から1, 2, 3番目のアイテム)

## ビルド

```sh
# Windows
./gradlew.bat build

# Linux
./gradlew build
```

ビルド & プレイは

```sh
# Windows
./start.ps1

# Linux
./start.sh
```
